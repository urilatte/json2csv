# Changelog

## [1.0.0] - 2026-06-20
### Añadido
- Esqueleto inicial del proyecto basado en lineamientos de Cookiecutter.
- Módulo `converter.py` con lógica robusta para control de excepciones en transformaciones de JSON a CSV.
- Configuración de pipeline automatizado CI/CD (.github/workflows/ci.yml).
- Suite de pruebas unitarias con Pytest alcanzando +85% de cobertura.
