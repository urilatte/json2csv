# Contexto del Proyecto: json2csv

## Propósito
Herramienta en Python para transformar archivos estructurados de formato JSON (listas de objetos planos) a formato CSV, asegurando controles robustos de excepciones.

## Arquitectura y Calidad
- Formateo: Black y Ruff (PEP8/PEP257).
- Tipado: Estricto con MyPy y PyRight.
- Seguridad: Validado con Bandit (No usar Trufflehog).
- Cobertura: Mínimo 85% con Pytest.
