# json2csv - Convertidor de formatos de datos

Herramienta desarrollada para la materia Calidad de Software (Universidad FASTA). Convierte archivos estructurados en formato JSON a formato CSV siguiendo lineamientos estrictos de integración continua.

## Características de Calidad e Integración
- **Formateo y Estilo:** Validado bajo estándares PEP8 usando Black y Ruff.
- **Documentación:** Docstrings bajo convención PEP257.
- **Tipado Estático:** Verificado estrictamente con MyPy y PyRight.
- **Pruebas:** Cobertura de tests automatizados superior al 85% con Pytest.
- **Seguridad:** Evaluado con Bandit sin observaciones críticas.
